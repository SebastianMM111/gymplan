GYM Tracker is a single-file, static workout tracker built for a 5-day Push/Pull/Legs/Upper/Lower split. No backend, no account, no install — just open and train.
Features:

Track sets, reps, and weight for every exercise
Rest timer with presets (60s / 90s / 2min / 3min)
Per-exercise personal notes
Weekly progress bars
3 themes: Dark, Purple, Kawaii
Reset Week with one click (keeps your profile)

Data: Everything saves locally to the user's browser via localStorage. Nothing is sent anywhere — each person's data stays on their own device.
Hosting: Pure HTML/CSS/JS — works on any static host including GitHub Pages. No dependencies, no build step
